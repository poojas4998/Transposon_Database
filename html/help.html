Search Transposon Class

To use the Search Class page, the user can simply click on the dropdown menu and select a transposon class to search by. Once a selection has been made, the submit button can be used to generate the results. To download a CSV file of the results, the Download as CSV button can be clicked. To save the image, the Download Image button can be clicked. To reset the page, the Reset button can be clicked.

Search Transposon Order

On the Search Transposon Order page, the user has the the ability to search transposon orders by species (left column of page) and also by the name of the transposon order itself (right column of page). When searching transposon orders by species, the user needs to select which species they would like transposon orders displayed for via a drop down menu. Then, the user needs to click the green submit button order to produce the pie chart and data table. The user can then download the corresponding table and pie chart using the "Download Table 1 CSV Information" & "Download Pie Chart 1" buttons. To reset the form, corresponding table, and pie chart, the user can click the red reset button.

When searching for a particular trasposon order, the user needs to enter the transposon order of their choice into a text box. By clicking the green submit button, a pie chart and data table summarizing the count of the transposon order in each species will be displayed. Like before, the user can then download the corresponding table and pie chart using the "Download Table 2 CSV Information" & "Download Pie Chart 2" buttons. To reset the form and corresponding table and pie chart, the user can click the red reset button.

Search Species

This particular page searches the database and provides the distribution of Transposon class, order, superfamily and length for a particular species that is selected in descending order. The drop down menu allows the user to select between the four different species of focus.
